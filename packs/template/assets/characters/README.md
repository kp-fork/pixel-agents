Optional character sprite set for this pack.

If you use `characterSpritesDir` in `manifest.json`, include all files:

- `char_0.png`
- `char_1.png`
- `char_2.png`
- `char_3.png`
- `char_4.png`
- `char_5.png`

Each file format must match the runtime expectation:

- 112x96 PNG
- 3 rows (down/up/right)
- 7 frames per row
- 16x32 per frame
